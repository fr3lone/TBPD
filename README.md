# TBPD
A compilation of finished tasks on spatial data  
Для проверки выполнения задания скачайте файл, содержащий код и перетащите его в рабочую область Jupyter Notebook.
Для корректной работы программы в правом верхнем углу вкладки установите рабочее ядро на Python 3.13 (XPython).
Затем поблочно запустите программу и проверьте правильность выводов.
